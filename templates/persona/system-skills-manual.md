## Available System Skills

These capabilities are available but **not active until invoked**.
After `/clear`, all skill context is removed — invoke again when needed.

| Skill | Capability |
|---|---|
| `/skills-sme` | Skill gap detection and creation guidance |
| `/memory-wf` | Memory file management |
| `/skills-wf` | Scaffold and create new skills |

> To make system skills always-on: invoke `/memory-wf` → "toggle system skills to always-on"
